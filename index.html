How it works

Simple behavior

The helper does one narrow thing.

1. Detects

Recognizes a supported Foxhole action or mine state.

2. Waits

Runs a timer or watches the mine inventory state.

3. Acts

Holds LMB or loads containers and transport when supported.

Works with

  • Hammer, Shovel, Sledge Hammer
  • Sulfur, Salvage, and Component mines
  • Container and transport loading from mines
  • Tested: 1920×1080, 2560×1440, and 3440×1440
  • Background mode with optional Telegram full-inventory notification

Does not do

  • No movement or aiming
  • No gameplay decisions
  • No generic rapid autoclicker mode
  • No support for every resolution yet
Setup

Install

  1. Download the installer.
  2. Run the installer.
  3. Run NFG A-Matic Player.
  4. Install Foxhole Helper from Catalog.
  5. Enable it and start Foxhole in windowed or borderless mode.
