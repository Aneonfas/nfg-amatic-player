How it works

Simple behavior

The helper does one narrow thing.

1. Detects

Recognizes a supported Foxhole action state.

2. Waits

Runs a short timer and shows progress.

3. Holds

Holds LMB while the supported state is active.

Works with

  • Hammer, Shovel, Sledge Hammer
  • Selected BMS facility actions
  • Windowed or borderless Foxhole
  • Tested: 1920×1080 and 3440×1440

Does not do

  • No movement or aiming
  • No gameplay decisions
  • No generic rapid autoclicker mode
  • No support for every resolution yet
Setup

Install

  1. Download the installer.
  2. Run the installer.
  3. Run NFG A-Matic Player.
  4. Install Foxhole Helper from Catalog.
  5. Enable it and start Foxhole in windowed or borderless mode.
