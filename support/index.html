Setup

Basic setup and checks.

Install

  1. Download the installer.
  2. Run the installer.
  3. Run NFG A-Matic Player.
  4. Open Catalog and install Foxhole Helper.
  5. Enable the package.

Check first

  • Use windowed or borderless Foxhole.
  • Use a tested screen profile when possible.
  • Set the Foxhole game UI language to Russian.
  • Windows may warn because the beta is not code-signed.
  • Download the installer, not Source code.

Bug report

How to report a bug

Send enough context to reproduce the issue.

  • Player version: 0.1.5-beta.1.
  • Windows version.
  • Screen resolution and Foxhole window mode.
  • Screenshot or short video of the problem.
  • Steps to reproduce.

Where to send it

Use Discord for support and bug reports. Create a post in #bug-reports for a specific bug, or ask in #support for quick help.

Log file

Attach the latest Player log when possible.

%LOCALAPPDATA%\OverlayAutomationStudio\product-player\logs\player-*.log
