Setup

Basic setup and checks.

Install

  1. Download the installer.
  2. Run the installer.
  3. Run NFG A-Matic Player.
  4. Open Catalog and install Foxhole Helper.
  5. Enable the package.

Check first

  • Use windowed or borderless Foxhole.
  • Use a tested screen profile when possible.
  • Windows may warn because the beta is not code-signed.
  • Download the installer, not Source code.
