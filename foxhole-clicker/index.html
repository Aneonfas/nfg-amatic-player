Foxhole Helper

A beta helper for Foxhole work actions, resource mines, and background mode.

Public beta package for NFG A-Matic Player.
Download installerSetup

Function

  • Detect supported action or mine state.
  • Wait timer or watch mine inventory state.
  • Hold LMB or load containers and transport.
  • Show status overlay and optionally notify Telegram.

Scope

  • Hammer, Shovel, Sledge Hammer.
  • Sulfur, Salvage, Component mines and selected BMS facility actions.
  • Tested screen profiles: 1920×1080, 2560×1440, 3440×1440.
  • Windowed or borderless mode; supported background mode.
  • This beta requires the Russian Foxhole game UI.

Not included

  • No movement.
  • No aiming.
  • No targeting.
  • No generic rapid clicker.
